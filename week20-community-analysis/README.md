# Week 20 — Community Analysis: Diversity and Ordination

## Learning objectives

- Organise abundance data for analysis
- Calculate alpha diversity indices (Shannon, Simpson, Chao1)
- Perform statistical tests on alpha diversity
- Compute beta diversity distances (Bray-Curtis)
- Run ordination analyses (PCoA, NMDS)
- Apply multiple testing correction

See the [Week 20 wiki page](https://github.com/GeoGenetics/NBIB25004U-Metagenomics-2026/wiki/Week-20).
